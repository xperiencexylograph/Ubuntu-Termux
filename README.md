# termux-ubuntu

Cara Install Dan Pasang Aplikasi Termux Di Android. 

Dengan kode perintah seperti di bawah ini.
- pkg update
- pkg upgrade
- pkg install git
- pkg install wget
- pkg install proot
- git clone https://github.com/odzayosbourne/Ubuntu
- cd ubuntu
- chmod +x ubuntu.sh
- ./ubuntu.sh

Kode untuk menjalankan Ubuntu di Termux : 
- ./start-ubuntu.sh
